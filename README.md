# GM - Gestion des Malades

## Overview
A JavaFX application for managing patients (Gestion des Malades). This project features a modern, elite UI with full system management capabilities including user registration, login, and profile management.

## Features
- **Modern UI**: Elite aesthetic inspired by high-end dashboard designs.
- **Auto-Maximized Launch**: Starts in full-screen mode for an immersive experience.
- **User Authentication**: Secure Login and Registration flow.
- **Profile Management**: Update user details and manage accounts.
- **Responsive Design**: Centered layouts that adapt to any screen size.

## Technologies Used
- Java 17
- JavaFX 17
- Maven
- MySQL
- CSS (Custom styling)

## How to Run
1. Ensure you have Java 17 and Maven installed.
2. Set up the MySQL database using the provided `setup.sql` script in `src/main/resources`.
3. Run the application using:
   ```bash
   mvn javafx:run
   ```
